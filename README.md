**A auto clicker built with .NET 8 and WPF.**

## Features

* Adjustable click speed (Clicks Per Second).
* Toggle seamlessly between left and right mouse buttons.
* Optional ±20% randomized timing interval variance to mimic organic input.
* Easily customizable hotkeys (default: `F6`).
* Supports both **Toggle** (press once to start/stop) and **Hold** (active only while key is held) modes.
* Optional audio feedback when toggling clicker state.
* Dark neon UI built with WPF.
* Zero external dependencies required to run the compiled build.
